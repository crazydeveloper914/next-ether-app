# Nexth

A Next.js + Ethereum starter kit to quickly ship Web3 Apps ⚡

![Nexth Readme Image](./readme.png)

## Packages 📦

- [App](./packages/app) - Next.js 13, with App router
- [Contracts](./packages/contracts/) - Hardhat projects

## Development 🛠️

```bash
npm run dev
# or
yarn dev
```
